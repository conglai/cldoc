# 文档命令

## 环境要求

* nodejs>=6.9.1: `node --version`

```
~ git clone git@gitlab.weixinzhuyi.com:wd/cl-toolkit.git
~ cd cl-toolkit
~ npm install --registry=https://registry.npm.taobao.org
~ npm link
```


## 命令列表

* `cl --help`: 帮助
* `cl doc`: 生成文档
